# 工程能力シミュレーター - 実装記録

## 概要
NV73S製品の工程能力をシミュレートするWebアプリケーション。
勤務時間、製品数量、非稼働時間を入力し、1日のスケジュールを可視化する。


## 主要機能

### 1. 入力パネル（左サイドバー）
- **勤務時間設定**: 開始・終了時刻
- **2Eモード**: 固定サイズ (85個/分, 2000個/巻)
- **混載モード**: 複数サイズの選択可能
- **Lot入力**: 最大10行、数量入力

### 2. 非稼働時間マネージャー
- 計画休憩 (60分固定)
- 設備立ち上げ (45分)
- 設備立ち下げ (10分)
- リール交換 (自動計算: 2分/回)
- Lot切替 (自動計算: 10分/回)
- 品種切替 (15分/回)

### 3. 結果表示（右パネル）
- スケジュール判定 (OK/超過)
- 終了予定時刻
- 純製造時間
- タイムライン可視化
- スケジュール詳細テーブル
- 非稼働時間内訳・割合
- 計算詳細 (PPH, PPM, 可動率)
